## 📥 Official Download & Security

As an independent project, we prioritize transparency. Currently, our builds are **not digitally signed** with Microsoft or Apple developer certificates. This may trigger security warnings during installation.

**Why is this happening?**
Certificates are expensive and require a lengthy verification process for indie developers. While we work on obtaining them, you can safely install the app by following the "Verified Source" instructions below.

### ⚠️ How to bypass Security Warnings

#### **🪟 Windows (Microsoft SmartScreen)**
When you run the `.exe` installer, you might see a blue window saying *"Windows protected your PC"*.
1. Click on **"More info"**.
2. Click the **"Run anyway"** button that appears.
*This happens because the app is new and hasn't established "reputation" with Microsoft yet.*

#### **🍎 macOS (Gatekeeper)**
When you try to open the app for the first time, macOS might say it *"cannot be opened because it is from an unidentified developer"*.
1. Open your **Applications** folder.
2. **Right-click** (or Control-click) the CoverCraft icon.
3. Select **Open** from the shortcut menu.
4. Click **Open** in the confirmation dialog.
*Simply double-clicking the icon will not work the first time.*

## 🔄 Stay Updated Automatically

CoverCraft includes a built-in **Auto-Updater**. Once you have installed the app manually for the first time, you won't need to visit this page again.
* The app checks for new versions on this repository every time it starts.
* If an update is found, it downloads in the background.
* You will be prompted to restart and apply the update instantly.
